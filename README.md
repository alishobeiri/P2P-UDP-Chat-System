### Client set up:
#### First run:
To intall the necessary dependencies, first use:
`npm install`

To run:
`cd backend`
`node client.js`

In another command window, to run frontend:
`npm run dev`

### Server Setup:
To run the server, run the server.js file on a remote public device. To do this, use:
`cd server`
`node server.js`
